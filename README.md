
# 🚀 ProManage – MERN Project Management App

ProManage is a full-stack **Project Management & Bug Tracking Application** built using the MERN stack.
It allows teams to create projects, manage tickets, assign tasks, and track progress using a Kanban workflow.

This project simulates real-world tools like Jira and Trello and demonstrates scalable SaaS architecture.

---

## ✨ Features

* 🔐 JWT-based User Authentication
* 📁 Create & Manage Projects
* 🎫 Create, Edit & Delete Tickets
* 👤 Assign Tickets to Team Members
* 🗂️ Kanban Board (To Do, In Progress, Done)
* 📊 Dashboard with Project & Ticket Overview
* 🔎 Filter & Search Tickets

---

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/Nyasa139/promanage.git
cd promanage
```

### Backend Setup

```bash
cd backend
npm install
npm start
```

### Frontend Setup

```bash
cd frontend
npm install
npm start
```

Create a `.env` file in the backend with:

```
MONGO_URI=your_mongodb_uri
JWT_SECRET=your_secret_key
PORT=5000
```
